\item Fertig

Nachdem die Virtuelle Maschine gebootet hat, kann die gewünschte Bildschirmauflösung ausgewählt werden.
Die Desktopicons bieten folgende Funktionalität:
\begin{itemize}
\item \textbf{Build \& Run} - Übersetzt den Kernel und führt das System virtualisiert aus.
\item \textbf{Eclipse} - Startet die vorkonfigurierte Entwicklungsumgebung
\item \textbf{Terminal} - Öffnet eine Terminalsession
\item \textbf{POTATOES folder} - Öffnet den Projektordner
